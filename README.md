# BizPro WordPress Theme

A professional, modern WordPress theme for business websites.

## Features

- ✅ Fully responsive design
- ✅ Custom post types (Services, Team, Testimonials)
- ✅ SEO optimized
- ✅ Performance optimized
- ✅ Modern code standards
- ✅ Contact form integration
- ✅ Google Maps support
- ✅ Portfolio/Projects showcase

## Installation

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme in WordPress admin
3. Customize via Appearance → Customize

## Development

Built with:
- HTML5, CSS3, JavaScript
- PHP 8.x
- WordPress 6.x best practices
